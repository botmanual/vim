编辑配置文件 vim ~/.vimrc
```
" ===== 注释自动换行 =====
set textwidth=80          " 每行最大80字符
set formatoptions+=t      " 自动换行文本
set formatoptions+=c      " 自动换行注释
set formatoptions+=q      " 允许gq格式化
set formatoptions+=r      " 回车自动延续注释
set formatoptions+=o      " o键自动延续注释

" ===== 显示更舒服 =====
set wrap                  " 自动显示换行
set linebreak             " 不在单词中间断行
syntax on              " 语法高亮
set number             " 显示行号
set hlsearch " 高亮显示搜索内容，nohlsearch关闭
```

如果是编辑模式，首先 esc 退出，然后依次按 gqap，g 是“全局/组合前缀”，q 代表“格式化”，a = around（围绕），p = paragraph
gq → 格式化
ap → 当前段落（paragraph）

也可以 gqq 只格式化当前一行（不用管段落），q 代表quicksingle line” 或 “line”（非官方，只是记忆辅助）

| 命令            | 作用                |
| ------------- | ----------------- |
| `:w`          | 保存当前文件            |
| `:w filename` | 保存到指定文件名          |
| `:wq`         | 保存 + 退出           |
| `:x`          | 保存 + 退出（等同 `:wq`） |
| `:q!`         | 不保存直接退出           |

向下查找（搜索）

按 Esc 确保在普通模式，然后输入：
/text_to_find
/ → 向下搜索
text_to_find → 你想找的内容
按 Enter 执行
光标会跳到第一个匹配结果
n → 下一处匹配
N → 上一处匹配

?text_to_find
? → 向上搜索
同样 n → 下一处匹配（在向上方向），N → 反向

:linenumber  快速跳转到某行

dd = delete line（删除整行）
Ndd 删除从光标所在行开始的 N 行：
5dd
删除当前行及下 4 行（总共 5 行）

:%d
% → 全文件范围
d → delete
结果：文件内容全清空，Vim 仍然打开
保存后再退出可写回空文件

D	删除光标所在行从光标到行尾
x	删除光标下一个字符
X	删除光标前一个字符

u	撤销删除
Ctrl + r	重做撤销

:s/old_text/new_text/
s → substitute（替换）
old_text → 要替换的内容
new_text → 新内容
只会替换光标所在行的第一个匹配

:s/foo/bar/g
g → 全局（行内所有匹配）

:%s/old_text/new_text/g
% → 全文件范围
s → 替换
g → 全局（每行所有匹配）

:%s/old_text/new_text/gc
c → confirm，每次替换前会提示
交互提示：
y → 替换这次
n → 跳过
a → 替换所有
q → 退出替换


通模式	直接按键	dd, gqap, n
命令行模式	: 开头	:w, :%s/old/new/g
